# The World Wide Web

## What is the World Wide Web?

* **Definition:** A system of interlinked, hypertext documents accessed via the Internet, enabling retrieval and display of text and media.
* **Inventor:** Tim Berners-Lee (1991), initially as a project at CERN.
* **Key Concept:** Hypertext - linked words/phrases within text that allow users to jump to other documents or sections.

## Evolution of the Web

* **Early Web:** Primarily text-based, with limited functionality.
* **Graphical Web:** Marc Andreessen's NCSA Mosaic browser introduced hypermedia (graphics, sound, video), driving rapid adoption.

## How the Web Works

### 1. Web Browsers

* **Function:** Software interface for accessing and viewing web content (text, images, videos, etc.).
* **Examples:** Internet Explorer, Mozilla Firefox, Google Chrome, Safari.
* **Capabilities:**  Surfing the web, searching, emailing, file transfer, and more.

### 2. Hypertext Markup Language (HTML)

* **Function:**  Programming language that structures web pages and creates hyperlinks.
* **Hyperlinks:** Clickable elements that direct users to other web pages or files.

### 3. Hypertext Transfer Protocol (HTTP)

* **Function:** Protocol that governs communication between web browsers and web servers.
* **Process:**
    * Browser sends an HTTP request to the server.
    * Server retrieves the requested web page and sends it back to the browser.
* **HTTPS:**  Secure version of HTTP, providing encryption for sensitive data.

### 4. File Transfer Protocol (FTP)

* **Function:** Protocol for transferring files between computers over the Internet.
* **Uses:** Downloading and uploading files, software, and other data.
